---




## 게임 소개

> 플레이어는 기사(Knight)를 조작해 장애물을 피하고, 아이템을 획득하며 최대한 오래 생존하는 것을 목표로 합니다.

- 체력이 서서히 감소하며, 회복 아이템으로 생존 시간 연장
- 장애물 충돌 시 체력 감소
- 코인 획득 시 점수 증가

## 주요 기능

| 기능 구분 | 설명 |
|-----------|------|
| 캐릭터 이동 | 점프 / 슬라이드 구현 |
| 장애물 시스템 | 랜덤 등장, 충돌 감지 |
| 아이템 시스템 | 체력 회복, 코인 획득 |
| UI | 시작, 인게임, 게임오버 화면 |
| GameManager | 점수 계산 및 게임 상태 관리 |
| 맵 반복 | GroundManager를 통한 무한 스크롤 |
